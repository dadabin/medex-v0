# MeDEX UI 样式规范

> 极简暗色界面，参照 OpenAI Codex 风格。克制、精密、高端。

---

## 1. 设计语言

**关键词：** 极简 · 克制 · 留白 · 高端

去掉一切装饰性元素。无彩色图标、无emoji、无渐变、无多余边框。内容本身就是界面。信息层级靠字号和灰度区分，不靠颜色。唯一的彩色点缀是交互态的 accent 绿。

---

## 2. 色彩

### 灰度体系（占界面 95%）

| Token | 色值 | 用途 |
|-------|------|------|
| `--bg-0` | `#0A0A0A` | 页面底色 |
| `--bg-1` | `#111111` | 侧边栏、面板 |
| `--bg-2` | `#1A1A1A` | 卡片、代码块、输入框 |
| `--bg-3` | `#222222` | Hover、Active |
| `--border` | `#1E1E1E` | 分割线（极弱） |
| `--border-subtle` | `#161616` | 更弱的分割 |
| `--t1` | `#F0F0F0` | 标题、正文主文字 |
| `--t2` | `#888888` | 辅助说明、时间 |
| `--t3` | `#555555` | 占位符、禁用 |
| `--t4` | `#333333` | 极弱文字 |

### 强调色（仅交互态）

| Token | 色值 | 用途 |
|-------|------|------|
| `--accent` | `#10B981` | 链接、按钮、Focus |
| `--accent-dim` | `rgba(16,185,129,0.12)` | Tag 背景、行高亮 |

### 语义色（极少使用，仅状态点）

| Token | 色值 | 用途 |
|-------|------|------|
| `--ok` | `#10B981` | 完成 |
| `--warn` | `#F59E0B` | 等待/确认 |
| `--err` | `#EF4444` | 失败 |
| `--run` | `#6B7280` | 执行中（灰色旋转） |

### 医学实体色（仅行内标注）

| 实体 | 文字色 | 背景 |
|------|--------|------|
| 疾病/诊断 | `#F0F0F0` | `rgba(255,255,255,0.06)` |
| 药物 | `#F0F0F0` | `rgba(255,255,255,0.06)` |
| 检查/检验 | `#F0F0F0` | `rgba(255,255,255,0.06)` |

> 实体标注不使用颜色区分，仅用 `1px` 下划线虚线标记 + Tooltip 区分类型。保持文字颜色统一。

## 3. 字体

| 用途 | 字体栈 |
|------|--------|
| 正文 | `"-apple-system", "Inter", "Noto Sans SC", sans-serif` |
| 代码 | `"SF Mono", "JetBrains Mono", "Menlo", monospace` |

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| `--t-h1` | `20px` | 600 | 1.3 | 页面标题 |
| `--t-h2` | `15px` | 600 | 1.4 | 区块标题 |
| `--t-body` | `14px` | 400 | 1.65 | 正文 |
| `--t-sm` | `12px` | 400 | 1.5 | 辅助文字 |
| `--t-xs` | `11px` | 500 | 1.4 | Badge |
| `--t-code` | `13px` | 400 | 1.55 | 代码 |

## 5. 圆角

| Token | 值 | 用途 |
|-------|-----|------|
| `--r-sm` | `4px` | Tag |

## 6. 布局

### 三栏

```
┌──────────────────────────────────────────────────────────────────┐
│  顶栏  h=48px                                                    │
│  MeDEX                                          [搜索]  [设置]    │
├────────┬─────────────────────────────────┬───────────────────────┤
│        │                                 │                       │
│ 左侧栏 │         对话区                   │  右侧文件面板          │
│ 240px  │         flex-1                   │  320px                │
│        │                                 │  ┌────┬──────────────┐│
│ 项目   │  消息流（含内联任务）              │  │树  │  文件预览     ││
│ 列表   │                                 │  │    │              ││
│        │                                 │  │    │              ││
│        │  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ │  │    │              ││
│        │  输入区                          │  │    │              ││
│        │                                 │  └────┴──────────────┘│
└────────┴─────────────────────────────────┴───────────────────────┘
```

### 关键规则

- 任务面板**不在顶部**，任务流内联在对话中
- 右侧文件面板分两栏：左侧文件树（80px）+ 右侧文件预览
- 分割线用 `1px solid var(--border)`，极弱，几乎看不见
- 消息区无最大宽度限制，左右 padding `48px`

## 7. 组件样式

### 7.1 消息

**用户消息：**
- 无气泡。左对齐。小字灰色 `--t2` 显示 "You"
- 内容直接渲染，`--t-body`，无背景无边框

**Agent 消息：**
- 无气泡。无左侧竖线。左对齐。小字灰色 `--t2` 显示 "MeDEX"
- 内容直接渲染
- 与用户消息之间仅 `24px` 间距分割

### 7.2 思考块

```
┌ thinking ───────────────────────────────┐
│  font: mono 13px  color: --t3           │
│  无背景 无边框 无图标                      │
│  默认折叠                                 │
└─────────────────────────────────────────┘
```

- 无彩色。纯灰色文字。
- 折叠为一行：`▸ thinking ...`，点击展开

### 7.3 工具调用（内联任务卡片）

```
▸ Bash · python analyze.py                ✅ 0.3s
  （点击展开）

展开后：
┌ Bash · python analyze.py ──────────── ✅ ─┐
│                                            │
│  $ python analyze.py --input data.json     │
│  font: mono 13px  color: --t2              │
│                                            │
│  诊断结果: 2型糖尿病                         │
│  font: mono 13px  color: --t3              │
│                                            │
└────────────────────────────────────────────┘
```

- 无彩色图标。工具名用 `--t2`，命令用 mono `--t3`
- 状态仅用极小圆点：`●` 绿/灰/红，`6px`
- 折叠态：单行，`14px` 行高
- 展开态：`bg-2` 背景，`1px solid border` 边框

### 7.4 任务组（内联在对话中）

```
┌ ▸ Task Plan ──────────────────────────── 1/4 ─┐
│                                                 │
│  1  ✓  分析主诉与病史                           │
│  2  ●  评估检查结果                             │
│  3     查询高血压用药指南                         │
│  4     生成诊断报告                              │
│                                                 │
│  ████████░░░░░░░░░░░░                           │
└─────────────────────────────────────────────────┘
```

- `bg-2` 背景，`1px border` 边框
- 状态：`✓` 绿色完成，`●` 灰色旋转执行中，空格待执行
- 无 emoji。无彩色文字。

### 7.5 需要补充信息

```
┌ 补充信息 ──────────────────────────────────────┐
│                                                 │
│  该患者是否进行过以下检查？                       │
│                                                 │
│  ○ 心电图                                       │
│  ○ 超声心动图                                   │
│  ● 以上均无                                     │
│                                                 │
│  [确认]                                         │
└─────────────────────────────────────────────────┘
```

- `bg-2` 背景，`1px border` 边框
- 标题用 `--t-sm` 大写，`--t3` 颜色
- 选项：单选按钮，`--t-body`
- 确认按钮：`bg-3` 背景，无边框，`--t1` 文字，hover 变 `accent`

### 7.6 文件链接

```
  analyze_report.md  ← 可点击，--accent 色，无图标
```

- 无 emoji 图标。纯文字链接，`--accent` 色
- hover 下划线

### 7.7 输入区

```
┌──────────────────────────────────────────────┐
│  bg-2  border  r-md                          │
│                                              │
│  输入医学问题...                     [发送 →]  │
│  font: 14px  color: --t1  placeholder: --t3  │
│  focus: border → --accent                    │
└──────────────────────────────────────────────┘
```

- 无上传按钮（上传在左侧栏或拖拽）
- 简洁的发送按钮：`→` 箭头，`--t3` 色，hover `--accent`

### 7.8 右侧文件面板

```
┌──────────────────────────────────────────────┐
│  文件                                   [上传] │
├──────┬───────────────────────────────────────┤
│ 📁   │  analyze_report.md                    │
│  src │                                       │
│   ├  │  # 患者分析报告                        │
│   ├  │                                       │
│   └  │  ## 基本信息                           │
│      │  - 姓名: 张三                          │
│ 上传 │  - 年龄: 58岁                          │
│      │                                       │
└──────┴───────────────────────────────────────┘
```

- 左侧文件树：`80px` 宽，仅图标+缩进，`--t3` 色
- 右侧预览：Markdown 渲染，`--t-body`
- 文件树选中项：`bg-3` 背景
- 无 emoji 图标。用 SVG 线条图标

### 7.9 侧边栏项目列表

```
┌────────────────────┐
│  + 新建             │  ← 简单文字按钮，--t2
│                    │
│  张三              │  ← --t1，无描述无图标
│  李四              │  ← --t3，非活跃
│  王五              │
│                    │
│  ─── 归档 ─────── │  ← --t4
│  赵六              │
└────────────────────┘
```

- 项目名仅一行文字。无描述、无图标、无元信息
- 活跃项：`--t1` + `bg-3` 背景
- 非活跃项：`--t3`

## 8. 图标

| 规则 | 说明 |
|------|------|
| 风格 | 线条 SVG，1.5px 描边 |
| 颜色 | 继承 `currentColor`，始终为 `--t2` 或 `--t3` |
| 尺寸 | `16px`，行内 `14px` |
| 禁止 | 无 emoji、无彩色图标、无填充图标 |

---

## 9. 动效

| 场景 | 效果 |
|------|------|
| 折叠/展开 | `height` 过渡 `150ms ease` |
| Hover | `background` 过渡 `100ms ease` |
| Focus | `border-color` 过渡 `100ms ease` |
| 流式光标 | `▍` 闪烁 `1s step-end infinite`，`--t3` 色 |
| 执行中 | 旋转环 `1s linear infinite`，`--t3` 色 |

> 无弹跳、无缩放、无渐变、无粒子。所有动效仅功能性。

---

## 10. 医学实体

行内标记：`1px dotted` 下划线，颜色 `--t3`。Hover 弹出 Tooltip 显示实体类型。不使用彩色背景和彩色文字。
